# manga-detail-app


To run this app follow the steps
1. Make a .env file containing following environment variables 
```
DB_USER=<db_username>
DB_PASSWORD=<db_password>
DB_SCHEMA=<db_schema>
```
2. Run the server via 
```
cd server/
node app.js
```
3. Run the client via
```
cd client/
npm start
```
